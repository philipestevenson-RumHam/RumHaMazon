How to use

  1. Show your bookmarks bar — press Ctrl+Shift+B (or Cmd+Shift+B on Mac)
  2. Drag the purple button up onto your bookmarks bar
  3. Go to any Amazon product page (works on all Amazon domains)
  4. Click the bookmark — an overlay appears with title, description and images, plus a Download Pack button
💡 Tip: The Download Pack button creates a ZIP file containing all product images (1.jpg, 2.jpg, …) and a CSV with the title and description. It requires a brief internet connection to load the zip library the first time.
